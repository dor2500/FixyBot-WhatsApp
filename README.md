# 🛠️ FixyBot - בוט תמיכה טכנית לטלגרם

בוט תמיכה טכנית חכם המבוסס על **Gemini AI** ומאגר ידע פנימי.  
הבוט מסוגל לנתח תמונות של תקלות, לחפש פתרונות ברשת, ולענות על שאלות טכניות בעברית.

## ✨ יכולות

- 💬 מענה על שאלות טכניות בעברית
- 📸 ניתוח תמונות של שגיאות ותקלות
- 📚 מאגר ידע פנימי עם אפשרות טעינה מחדש
- 🔍 חיפוש בגוגל כגיבוי לשאלות שאינן במאגר
- 🕐 מודעות לזמן ומיקום המשתמש
- ✅ אימות קיום מוצרים לפני מתן תשובה

## 🚀 פריסה ב-Render.com

1. העלה את הקוד ל-GitHub
2. היכנס ל-[Render.com](https://render.com) וצור חשבון
3. לחץ **New → Web Service**
4. חבר את ה-repository מ-GitHub
5. הוסף את משתני הסביבה:
   - `TELEGRAM_BOT_TOKEN`
   - `GEMINI_API_KEY`
   - `KB_URL`
6. לחץ **Deploy** 🎉

## 🖥️ הרצה מקומית

```bash
# התקנת תלויות
pip install -r requirements.txt

# יצירת קובץ .env עם המפתחות
cp .env.example .env

# הרצה
python bot.py
```

## 📁 מבנה הפרויקט

```
FixyBot/
├── bot.py              # הקוד הראשי של הבוט
├── kb_manager.py       # ניהול מאגר הידע
├── requirements.txt    # תלויות Python
├── Dockerfile          # הגדרות Docker לפריסה
├── render.yaml         # הגדרות Render.com
└── .gitignore          # קבצים מוחרגים מ-Git
```
